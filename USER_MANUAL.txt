❌ 자주 발생하는 문제들:

🔴 MySQL 연결 실패:
문제: "Can't connect to MySQL server"
해결: 
1. XAMPP 실행 확인
2. MySQL 서비스 시작
3. 포트 24 사용 확인
4. 방화벽 설정 확인

명령어 확인:
/Applications/XAMPP/xamppfiles/bin/mysql -h localhost -P 24 -u root -e "SELECT 1"

🔴 Python 모듈 없음:
문제: "ModuleNotFoundError"
해결:
pip3 install mysql-connector-python aiohttp youtube-transcript-api playwright

🔴 Playwright 브라우저 없음:
문제: "Browser not found"
해결:
playwright install

🔴 API 키 오류:
문제: "Invalid API key"
해결:
1. 환경변수 설정 확인
2. API 키 유효성 검증
3. 사용량 한도 확인

🔴 크롤링 차단:
문제: "403 Forbidden" 또는 "봇 감지"
해결:
1. 요청 간격 늘리기 (0.5초 이상)
2. User-Agent 변경
3. 프록시 사용 고려
4. 헤드리스 모드 해제

🔴 메모리 부족:
문제: "Memory error"
해결:
1. 동시 처리 수 줄이기
2. 데이터 청크 단위 처리
3. 가비지 컬렉션 강제 실행
